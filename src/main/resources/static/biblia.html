Iron Progress Bar

Версия 1.0 Стабильная

📥 Установка

1. Скачайте файл progress.iron (кнопка внизу)
2. Поместите файл в папку с вашим проектом
3. Добавьте в начало вашего кода:

exec(open("progress.iron").read(), globals())

🎯 Доступные функции

Функция Параметры Описание
bar_progress() max: int Создаёт интерактивный прогресс-бар в терминале с анимацией и процентом выполнения.
help_progress() Нет Выводит справку по библиотеке.
📖 Пример использования:
exec(open("progress.iron").read(), globals())
bar_progress(50)
help_progress()
💡 Требуется: pip install rich

Iron Krypto Gram

📥 Установка

1. Скачайте файл kripto_gram.iron

Функция Параметры Описание

search_coin() keyword: str Поиск монет по названию или символу.
help_krypto() Нет Справка по библиотеке.
📖 Пример:
exec(open("kripto_gram.iron").read(), globals())
search_coin("bitcoin")
get_coin(90)

Iron Grafic

Версия 1.0 Новая

📥 Установка

1. Скачайте файл grafic.iron (кнопка внизу)
2. Поместите файл в папку с вашим проектом
3. Добавьте в начало вашего кода:

exec(open("grafic.iron").read(), globals())

🎯 Доступные функции

Функция Параметры Описание
e_graf() x: list, y: list, title: str Создаёт интерактивный линейный график. Поддерживает масштабирование, наведение для просмотра значений, экспорт в PNG. Идеально для визуализации трендов, временных рядов и математических функций.
circle_diogram() values: list, names: list, title: str Создаёт круговую диаграмму с настраиваемыми цветами и легендой. Автоматически рассчитывает проценты. Подходит для отображения долей, распределения данных и статистики.
help_grafic() Нет Выводит подробную справку со всеми доступными функциями, параметрами и примерами использования.
📖 Примеры использования:
# Активация библиотеки
exec(open("grafic.iron").read(), globals())

# 1. Линейный график
e_graf(x=[1, 2, 3, 4, 5], y=[10, 15, 13, 17, 20], title="Рост продаж")

# 2. Круговая диаграмма
circle_diogram(
values=[35, 25, 20, 15, 5],
names=["Поиск", "Соцсети", "Реклама", "Email", "Другое"],
title="Источники трафика"
)

# 3. Визуализация данных из файла
data = [10, 25, 30, 45, 60, 55, 70]
months = ["Янв", "Фев", "Мар", "Апр", "Май", "Июн", "Июл"]
e_graf(x=months, y=data, title="Динамика за полгода")

# 4. Справка
help_grafic()
💡 Возможности графиков Plotly:
• Интерактивное масштабирование (колёсико мыши)
• Наведение курсора для просмотра точных значений
• Панорамирование (перетаскивание графика)
• Сохранение графика как PNG (кнопка в интерфейсе)
• Автоматическая адаптация под данные
• Поддержка больших массивов данных (100 000+ точек)
📊 Требования: pip install plotly
💡 Совет: Графики открываются в браузере автоматически. Для работы в Jupyter используйте fig.show(renderer="notebook").
