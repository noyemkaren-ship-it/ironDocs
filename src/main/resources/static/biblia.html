⏳ Iron Progress Bar

Версия 1.0 Стабильная

📥 Установка

1. Скачайте файл progress.iron (кнопка внизу)
2. Поместите файл в папку с вашим проектом
3. Добавьте в начало вашего кода:

exec(open("progress.iron").read(), globals())

🎯 Доступные функции

Функция Параметры Описание
bar_progress() max: int (количество шагов) Создаёт интерактивный прогресс-бар в терминале. Автоматически показывает процент выполнения, анимированный индикатор и оценочное время завершения. Идеально подходит для отображения хода загрузки файлов, обработки данных или любых длительных операций.
help_progress() Нет Выводит в консоль подробную справку по работе с библиотекой: примеры использования, описание всех функций и советы по настройке.
📖 Пример использования:
# Активация библиотеки
exec(open("progress.iron").read(), globals())

# Простой прогресс-бар на 50 шагов
put("Начинаем загрузку данных...")
bar_progress(50)
put("Загрузка завершена!")

# Прогресс-бар внутри цикла
for i in range(100):
bar_progress(i)
sleep(0.01)

# Справка
help_progress()
💡 Примечание: Для работы библиотеки требуется установленный Python пакет rich. Установка: pip install rich

₿ Iron Krypto Gram

Версия 1.0 Стабильная

📥 Установка

1. Скачайте файл kripto_gram.iron (кнопка внизу)
2. Поместите файл в папку с вашим проектом
3. Добавьте в начало вашего кода:

exec(open("kripto_gram.iron").read(), globals())

🎯 Доступные функции

Функция Параметры Описание
all_coins() Нет Получает и отображает список ВСЕХ криптовалют с биржи. Для каждой монеты показывает: название (symbol), текущую цену в USD, уникальный идентификатор (ID). Возвращает полный JSON-ответ от API CoinLore для дальнейшей обработки. Всего доступно более 14 000+ монет.
get_coin() id: int (ID монеты) Получает детальную информацию о конкретной монете по её уникальному ID. Возвращает: текущую цену, рыночную капитализацию, объём торгов за 24 часа, изменение цены в процентах (1ч, 24ч, 7д), максимальную и минимальную цену за всё время, оборотное предложение и другие метрики. Пример: get_coin(90) — информация о Bitcoin.
get_coin_oper_24h() coin_id: int (ID монеты) Анализирует исторические данные монеты за последние 24 часа. Возвращает OHLCV данные: Open (цена открытия), High (максимум), Low (минимум), Close (цена закрытия), Volume (объём торгов). Идеально подходит для технического анализа, построения графиков и определения трендов. Данные разбиты по 4-часовым интервалам.
search_coin() keyword: str (поисковый запрос) Ищет криптовалюты по названию (name) или символу (symbol). Поддерживает частичное совпадение (например, "coin" найдёт Bitcoin, Dogecoin и другие). Выводит все найденные совпадения с их ценой и ID. Возвращает список найденных монет. Пример: search_coin("bit") — найдёт Bitcoin, BitTorrent, BitShares.
help_krypto() Нет Выводит в консоль полную документацию по библиотеке: список всех функций с описанием, ID популярных монет (Bitcoin: 90, Ethereum: 80, Doge: 2, XRP: 58, Cardano: 14, Solana: 161), примеры использования и советы по работе с API.
📖 Примеры использования:
# Активация библиотеки
exec(open("kripto_gram.iron").read(), globals())

# 1. Показать все монеты
all_coins()

# 2. Поиск Bitcoin по названию
search_coin("bitcoin")

# 3. Получить информацию о Dogecoin (ID = 2)
doge = get_coin(2)
put(doge)

# 4. Анализ истории Ethereum за 24ч (ID = 80)
history = get_coin_oper_24h(80)
for candle in history:
put(f"Open: {candle['open']}, Close: {candle['close']}")

# 5. Справка
help_krypto()
