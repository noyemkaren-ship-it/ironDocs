О языке Iron

Iron — это современный, удобный и сверхпростой язык программирования, написанный на Python 3. Он создан для того, чтобы любой человек — от школьника до профессионального разработчика — мог с лёгкостью писать программы, тестировать API, создавать GUI-приложения и работать с базами данных.

Iron объединяет в себе простоту Python, скорость C++ и мощь современной экосистемы. С Iron вы можете:

🚀 Мгновенный старт

Первая программа за 5 секунд. Никаких сложных конструкций.

🌐 HTTP из коробки

GET, POST и массовые запросы одной строкой. C++ под капотом для рекордной скорости.

🎨 GUI за 3 строки

Создавайте окна, кнопки, поля ввода. Поддерживается CustomTkinter и любой Python GUI.

🐍 Python-мост

Вызывайте ЛЮБОЙ Python-код прямо из Iron. Все 450 000+ библиотек PyPI — ваши.

💾 Базы данных

Встроенная поддержка SQLite. SELECT, INSERT, DELETE — прямо в коде.

⚡ C++ Модули

Тяжёлые задачи выполняются на C++. Массовые запросы — до 500 запросов в секунду.

Переменные

Объявление переменных в Iron — проще некуда. Используйте оператор %=%.

i %=% 1
name %=% Iron
pi %=% 3.14

При выводе переменной через put интерпретатор покажет её значение:

put i      // Выведет: 1
put name   // Выведет: Iron

Типы данных: Iron автоматически определяет тип (целое, дробное, строка). Можно явно указать int или str:

int age %=% 25
str city %=% Москва

PUT — Вывод

Оператор put нужен для вывода чего-либо на экран. Никаких скобок или кавычек!

put Hello World
put Привет, Iron!

Это выведет текст прямо в консоль. Сравните с Python: print("Hello World"). Iron — проще.

REQ / PREQ / REQS — HTTP запросы

Iron создавался как язык для тестирования API, поэтому HTTP-запросы — его суперсила.

REQ — GET запрос

Отправляет один GET-запрос:

req https://api.github.com

Ответ сервера будет выведен в консоль.

REQS — Массовые GET запросы (C++)

Самая мощная функция Iron! Запросы отправляются через скомпилированный C++ модуль с невероятной скоростью:

https://api.local/test
reqs

Интерпретатор спросит количество запросов. C++ отправит их со скоростью 500+ запросов/сек.

PREQ — POST запрос

Отправляет POST-запрос с JSON:

preq %=% {"name": "Iron", "version": "3.8"}

URL указывается отдельно через команду http или просто вставляется в любом месте скрипта.

MEMORY — Память

Оператор memory показывает все переменные и функции, которые сейчас находятся в памяти интерпретатора:

memory

Это незаменимый инструмент для отладки!

IF / ELSE — Условия

Iron поддерживает все основные операторы сравнения: =, >, <, >=, <=, !=.

Для указания тела условия используются префиксы:

  • %i% — выполняется, если условие ИСТИНА
  • %e% — выполняется, если условие ЛОЖЬ
i %=% 1

if i = 1
%i% put Переменная равна 1
%e% put Переменная не равна 1
Сравните с Python: 5 строк кода. В Iron — 3 строки! 🎯

%p% / %py%: / %py%/ — Python внутри Iron

Если вам не хватает встроенных функций Iron, вы можете вызвать ЛЮБУЮ Python-команду или библиотеку!

%p% — одна Python-команда

%p% print('Hello from Python!')
%p% btn = ctk.CTkButton(root, text="Кнопка")

%py%: / %py%/ — блок Python-кода

Позволяет выполнить целый блок Python-кода с правильными отступами:

%py%:
def calculate():
    a = 10
    b = 20
    print(f"Сумма: {a + b}")
%py%/

Это делает Iron бесконечно расширяемым! Все 450 000+ библиотек PyPI доступны вам.

GUI — Графический интерфейс

Iron — один из немногих языков, где GUI можно создать за 3 строки!

Создание окна

init_root Мой проект
lbl Привет, мир!
close

Кнопки и поля ввода

init_root Калькулятор

lbl Введите число:
%p% entry = ctk.CTkEntry(root)
%p% entry.pack()

%py%:
def submit():
    print(entry.get())
%py%/

btn Отправить % %p%submit()
close

Важно: Всегда завершайте GUI-скрипт командой close!

WHILE_E — Циклы

Повторяет команду указанное количество раз:

while_e 5 % put Повторяю!

// Или для запросов:
while_e 10 % req https://api.local/test

Первый аргумент — количество повторений, после % — команда.

FUN — Функции

Создавайте собственные функции с помощью оператора fun:

fun hello % put Привет, мир!
hello  // Вызовет функцию

Функции сохраняются в памяти и могут быть вызваны в любой момент.

READ — Ввод данных

Iron поддерживает пользовательский ввод:

read name
put Привет, name!

// Числовой ввод:
int read age
put Тебе age лет

LEX_F / LEX_S — Комментарии

В Iron комментарии работают через отключение лексера (валидатора команд):

lex_f
Это комментарий. Всё что угодно!
Лексер выключен, код не выполняется.
lex_s
put А это снова код!

LEX_F = LEXer False (выключить), LEX_S = LEXer Start (включить).

IMPORT / START — Модули

Iron поддерживает модульную структуру:

import tests/health_check.iron
start tests/main.iron

import подключает внешний .iron файл, start запускает его как отдельный скрипт.

MATH — Математика

Iron умеет считать прямо из коробки:

math 5 + 3    // 5 + 3 = 8
math 10 * 7   // 10 * 7 = 70
math 100 / 4  // 100 / 4 = 25
math 50 - 20   // 50 - 20 = 30

SLEEP — Пауза

Приостанавливает выполнение скрипта на указанное количество секунд:

put Ждём...
sleep 5
put Прошло 5 секунд!

@SAVE — Логирование

Сохраняет результат выполнения команды в файл scr/save.txt:

@save
put Это сохранится в лог!
