📝 Правила синтаксиса Iron

🎨 Подсветка синтаксиса — не ошибка!

Красная подсветка в редакторе — это НЕ ОШИБКА, а просто визуальное выделение синтаксиса! Iron использует цветовую схему, где:

  • Фиолетовый — ключевые слова (def, if, else, for, while, import)
  • Синий — функции (put, read, init_root, lbl, btn, inp, get_inp)
  • Зелёный — строки в кавычках
  • Серый курсив — комментарии (#)
  • Оранжевый — числа

Важно: Если код написан правильно, он будет работать! Красный цвет — это просто стиль, как в любом современном редакторе кода.

Базовые правила синтаксиса

# Комментарии начинаются с решётки
name = "Iron"  # Переменные не требуют указания типа

# Функции вызываются с круглыми скобками
put(name)

# Блоки кода отделяются отступами (4 пробела или табуляция)
if name == "Iron":
    put("Привет, Iron!")
    
# Циклы и условия обязательно заканчиваются двоеточием
for i in range(3):
    put(i)

🎯 put() — главная команда вывода в Iron

Главное преимущество put() — возвращает значение, а не просто выводит!

🐍 Python: print()

# print возвращает None
name = "John"
result = print(name)
# result = None ❌

# Нельзя использовать в выражениях
# x = print("text") + 5  — Ошибка!

⚡ Iron: put()

# put ВОЗВРАЩАЕТ текст!
name = "John"
result = put(name)
# result = "John" ✅

# Можно использовать цепочки!
x = put("Iron") + " rocks!"
put(x)  # "Iron rocks!"

# Создание переменной с выводом одной строкой
user = put("Карен")  # Выведет "Карен" И сохранит в user!

Гениальная фишка Iron: name = put("Карен")

# В Iron можно так! Это и вывод, и создание переменной!
user = put("Карен")
# Вывод: Карен
# Переменная user = "Карен"

# Реальный пример:
greeting = put("Добро пожаловать в Iron!")
put(greeting)  # Выведет "Добро пожаловать в Iron!"

# Можно цепочкой:
put(put("Привет") + ", мир!")
# Сначала выведет "Привет", потом "Привет, мир!"

⚖️ Условные операторы в Iron

if/elif/else

score = 85

if score >= 90:
    put("Отлично! 5")
elif score >= 75:
    put("Хорошо! 4")
elif score >= 60:
    put("Удовлетворительно! 3")
else:
    put("Нужно подтянуть знания!")

Упрощённый if_e()

# Синтаксис: if_e(что1, оператор, что2, код)
age = 18
if_e(age, ">=", 18, 'put("Доступ разрешён")')

# Доступные операторы: =, ==, >, <, >=, <=, !=
if_e(user_input, "==", "admin", 'put("Добро пожаловать, администратор!")')

🔄 Циклы в Iron

for — перебор элементов

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    put(fruit)

for i in range(5):
    put(f"Итерация {i}")

while — с условием

counter = 0
while counter < 5:
    put(f"Счётчик: {counter}")
    counter += 1

Упрощённый while_e()

while_e(3, 'put("Iron rocks!")')

count = 0
while_e(5, 'count += 1; put(f"Шаг {count}")')

📁 Работа с файлами в Iron

Iron поддерживает все операции с файлами через стандартные Python функции.

Чтение файла

# Чтение всего файла
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
    put(content)

# Чтение построчно
with open("file.txt", "r") as f:
    for line in f:
        put(line.strip())

Запись в файл

# Запись (перезапись)
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello from Iron!")

# Дозапись в конец
with open("log.txt", "a") as f:
    f.write("Новая строка\n")

Проверка существования файла

import os
if os.path.exists("file.txt"):
    put("Файл существует!")
else:
    put("Файл не найден")

🪟 Графический интерфейс (GUI)

Iron имеет встроенную поддержку GUI через customtkinter. Создание окон — в 2 строки кода!

Функции для работы с GUI

ФункцияОписаниеПример
init_root(title)Создаёт окноwin = init_root("Моё окно")
lbl(parent, text)Текстовая меткаlbl(win, "Привет!")
btn(parent, text, command)Кнопкаbtn(win, "Нажми", func)
inp(parent, placeholder)Поле вводаentry = inp(win, "Введите текст")
get_inp(entry)Получить текстtext = get_inp(entry)

Пример: Кликер

win = init_root("Кликер")

clicks = 0
label = lbl(win, f"Счёт: {clicks}")

def on_click():
    global clicks
    clicks += 1
    label.configure(text=f"Счёт: {clicks}")

btn(win, "Кликни меня!", on_click)
win.mainloop()

🐧 Запуск Iron файлов из терминала

Способ 1: Алиас в .bashrc (простой и быстрый)

# Открой файл .bashrc
nano ~/.bashrc

# Добавь в конец файла строку:
alias iron='python3 /путь/к/вашему/main.py'

# Например, если проект в ~/Iron-Langhute:
alias iron='python3 ~/Iron-Langhute/main.py'

# Сохрани и примени изменения:
source ~/.bashrc

# Теперь можно запускать:
iron my_program.iron

Способ 2: Создание исполняемого скрипта (профессиональный)

# Создай файл в /usr/local/bin
sudo nano /usr/local/bin/iron

# Вставь в него:
#!/bin/bash
python3 /путь/к/вашему/main.py "$@"

# Сделай файл исполняемым:
sudo chmod +x /usr/local/bin/iron

# Теперь команда iron работает глобально!
iron script.iron

Способ 3: Добавление в PATH

# Создай папку для скриптов
mkdir -p ~/.local/bin

# Создай скрипт iron
nano ~/.local/bin/iron

# Содержимое:
#!/usr/bin/env python3
import sys
from functions import *
with open(sys.argv[1]) as f:
    exec(f.read(), globals())

# Сделай исполняемым и добавь в PATH
chmod +x ~/.local/bin/iron
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
