📝 Правила синтаксиса Iron

🎨 Подсветка синтаксиса — не ошибка!

Красная подсветка в редакторе — это НЕ ОШИБКА, а просто визуальное выделение синтаксиса! Iron использует цветовую схему, где:

  • Фиолетовый — ключевые слова (def, if, else, for, while, import)
  • Синий — функции (put, read, init_root, lbl, btn, inp, get_inp)
  • Зелёный — строки в кавычках
  • Серый курсив — комментарии (#)
  • Оранжевый — числа

Важно: Если код написан правильно, он будет работать! Красный цвет — это просто стиль, как в любом современном редакторе кода.

Базовые правила синтаксиса

# Комментарии начинаются с решётки
name = "Iron"  # Переменные не требуют указания типа

# Функции вызываются с круглыми скобками
put(name)

# Блоки кода отделяются отступами (4 пробела или табуляция)
if name == "Iron":
    put("Привет, Iron!")
    
# Циклы и условия обязательно заканчиваются двоеточием
for i in range(3):
    put(i)

🎯 put() — главная команда вывода в Iron

Главное преимущество put() — возвращает значение, а не просто выводит!

🐍 Python: print()

# print возвращает None
name = "John"
result = print(name)
# result = None ❌

# Нельзя использовать в выражениях
# x = print("text") + 5  — Ошибка!

⚡ Iron: put()

# put ВОЗВРАЩАЕТ текст!
name = "John"
result = put(name)
# result = "John" ✅

# Можно использовать цепочки!
x = put("Iron") + " rocks!"
put(x)  # "Iron rocks!"

# Создание переменной с выводом одной строкой
user = put("Карен")  # Выведет "Карен" И сохранит в user!

Гениальная фишка Iron: name = put("Карен")

# В Iron можно так! Это и вывод, и создание переменной!
user = put("Карен")
# Вывод: Карен
# Переменная user = "Карен"

# Реальный пример:
greeting = put("Добро пожаловать в Iron!")
put(greeting)  # Выведет "Добро пожаловать в Iron!"

# Можно цепочкой:
put(put("Привет") + ", мир!")
# Сначала выведет "Привет", потом "Привет, мир!"

⚖️ Условные операторы в Iron

if/elif/else

score = 85

if score >= 90:
    put("Отлично! 5")
elif score >= 75:
    put("Хорошо! 4")
elif score >= 60:
    put("Удовлетворительно! 3")
else:
    put("Нужно подтянуть знания!")

Упрощённый if_e()

# Синтаксис: if_e(что1, оператор, что2, код)
age = 18
if_e(age, ">=", 18, 'put("Доступ разрешён")')

# Доступные операторы: =, ==, >, <, >=, <=, !=
if_e(user_input, "==", "admin", 'put("Добро пожаловать, администратор!")')

🔧 Функции в Iron

Определение функций

# Базовая функция
def greet(name):
    return put(f"Привет, {name}!")

# Функция с несколькими параметрами
def calculate(a, b):
    result = a + b
    put(f"{a} + {b} = {result}")
    return result

# Вызов функций

sum_value = calculate(5, 3)

📁 Работа с файлами в Iron

Iron поддерживает все операции с файлами через стандартные Python функции.

Чтение файла

# Чтение всего файла
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
    put(content)

# Чтение построчно
with open("file.txt", "r") as f:
    for line in f:
        put(line.strip())

Запись в файл

# Запись (перезапись)
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello from Iron!")

# Дозапись в конец
with open("log.txt", "a") as f:
    f.write("Новая строка\n")

Проверка существования файла

import os
if os.path.exists("file.txt"):
    put("Файл существует!")
else:
    put("Файл не найден")

🆕 Новые функции в Iron 2.0

NEW read_file() — чтение файла в список строк

Функция read_file() читает файл и возвращает список строк. Это намного проще, чем использовать стандартные методы Python!

# Синтаксис: read_file(имя_файла)
# Возвращает список строк файла

# Пример 1: Чтение файла и вывод всех строк
file_lines = read_file("save.txt")
for line in file_lines:
    put(line)

# Пример 2: Чтение конфигурационного файла
config = read_file("config.ini")
put(f"Конфигурация загружена! {len(config)} строк")

# Пример 3: Обработка данных из файла
data = read_file("numbers.txt")
total = 0
for num in data:
    total += int(num)
put(f"Сумма: {total}")

📌 Преимущества read_file()

  • Автоматически открывает и закрывает файл
  • Возвращает готовый список строк
  • Работает с кодировкой UTF-8
  • Игнорирует пустые строки в конце файла

NEW sleep() — пауза в программе

Функция sleep() приостанавливает выполнение программы на указанное количество секунд.

# Синтаксис: sleep(секунды)

# Пример 1: Простая пауза
put("Загрузка...")
sleep(2)
put("Готово!")

# Пример 2: Таймер с обратным отсчётом
for i in range(5, 0, -1):
    put(i)
    sleep(1)
put("ПУСК! 🚀")

# Пример 3: Анимация загрузки
put("Установка Iron...")
for i in range(10):
    put(".", end="")
    sleep(0.5)
put("\n✅ Установка завершена!")

NEW rand() — генерация случайных чисел

Функция rand() возвращает случайное число в заданном диапазоне (включая границы).

# Синтаксис: rand(минимум, максимум)
# Возвращает случайное целое число от минимум до максимум

# Пример 1: Простая игра "Угадай число"
if rand(1, 3) == 1:
    put("Вы победили! 🎉")
else:
    put("Вы проиграли :(")

# Пример 2: Генерация пароля
password = ""
for i in range(8):
    char_code = rand(65, 90)  # Коды заглавных букв
    password += chr(char_code)
put(f"Ваш пароль: {password}")

# Пример 3: Бросание кубика
def roll_dice():
    return rand(1, 6)

put(f"Выпало: {roll_dice()}")

# Пример 4: Случайный выбор из списка
fruits = ["яблоко", "банан", "апельсин"]
random_index = rand(0, len(fruits) - 1)
put(f"Сегодня едим: {fruits[random_index]}")

🪟 Графический интерфейс (GUI) — Подробное руководство

Iron имеет встроенную поддержку GUI через customtkinter. Создание окон — невероятно просто!

🚀 Быстрый старт GUI

Всего 2 строки кода для создания окна с надписью:

win = init_root("Моё первое окно")
lbl(win, "Привет, Iron!")
win.mainloop()

Полный список функций GUI

Функция Описание Параметры Возвращает
init_root(title) Создаёт главное окно приложения title — заголовок окна Объект окна
lbl(parent, text) Создаёт текстовую метку parent — родительский элемент
text — текст метки		 Объект метки
btn(parent, text, command) Создаёт кнопку parent — родительский элемент
text — текст на кнопке
command — функция-обработчик		 Объект кнопки
inp(parent, placeholder) Создаёт поле ввода parent — родительский элемент
placeholder — текст-подсказка		 Объект поля ввода
get_inp(entry) Получает текст из поля ввода entry — объект поля ввода Строка с введённым текстом

Пример 1: Простой кликер

win = init_root("Кликер")

clicks = 0
label = lbl(win, f"Счёт: {clicks}")

def on_click():
    global clicks
    clicks += 1
    label.configure(text=f"Счёт: {clicks}")

btn(win, "Кликни меня!", on_click)
win.mainloop()

Пример 2: Калькулятор

win = init_root("Калькулятор")

lbl(win, "Введите два числа:")

num1_entry = inp(win, "Первое число")
num2_entry = inp(win, "Второе число")

result_label = lbl(win, "Результат: ")

def calculate():
    num1 = float(get_inp(num1_entry))
    num2 = float(get_inp(num2_entry))
    result = num1 + num2
    result_label.configure(text=f"Результат: {result}")

btn(win, "Сложить", calculate)
win.mainloop()

Пример 3: Игра "Угадай число" с GUI

win = init_root("Угадай число")

secret_number = rand(1, 10)
attempts = 0

lbl(win, "Я загадал число от 1 до 10")
guess_entry = inp(win, "Твой вариант")
message_label = lbl(win, "")

def check_guess():
    global attempts
    guess = int(get_inp(guess_entry))
    attempts += 1
    
    if guess == secret_number:
        message_label.configure(text=f"🎉 Угадал за {attempts} попыток!")
    elif guess < secret_number:
        message_label.configure(text="📈 Больше!")
    else:
        message_label.configure(text="📉 Меньше!")

btn(win, "Проверить", check_guess)
win.mainloop()

⚠️ Важные правила GUI

  • Всегда вызывайте win.mainloop() в конце программы
  • Функции-обработчики кнопок должны быть определены ДО создания кнопки
  • Для обновления текста используйте .configure(text="новый текст")
  • Не забывайте про global при изменении внешних переменных в функциях

🐧 Запуск Iron файлов из терминала

Способ 1: Алиас в .bashrc (простой и быстрый)

# Открой файл .bashrc
nano ~/.bashrc

# Добавь в конец файла строку:
alias iron='python3 /путь/к/вашему/main.py'

# Например, если проект в ~/Iron-Langhute:
alias iron='python3 ~/Iron-Langhute/main.py'

# Сохрани и примени изменения:
source ~/.bashrc

# Теперь можно запускать:
iron my_program.iron

Способ 2: Создание исполняемого скрипта (профессиональный)

# Создай файл в /usr/local/bin
sudo nano /usr/local/bin/iron

# Вставь в него:
#!/bin/bash
python3 /путь/к/вашему/main.py "$@"

# Сделай файл исполняемым:
sudo chmod +x /usr/local/bin/iron

# Теперь команда iron работает глобально!
iron script.iron

Способ 3: Добавление в PATH

# Создай папку для скриптов
mkdir -p ~/.local/bin

# Создай скрипт iron
nano ~/.local/bin/iron

# Содержимое:
#!/usr/bin/env python3
import sys
from functions import *
with open(sys.argv[1]) as f:
    exec(f.read(), globals())

# Сделай исполняемым и добавь в PATH
chmod +x ~/.local/bin/iron
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
