О языке Iron

Iron — это современный, удобный и сверхпростой язык программирования, написанный на Python 3. Он создан для того, чтобы любой человек — от школьника до профессионального разработчика — мог с лёгкостью писать программы, тестировать API, создавать GUI-приложения и работать с базами данных.

🚀 Мгновенный старт

Первая программа за 5 секунд. Никаких сложных конструкций.

🌐 HTTP из коробки

GET, POST и массовые запросы одной строкой. C++ под капотом.

🎨 GUI за 3 строки

Окна, кнопки, поля ввода. CustomTkinter и любой Python GUI.

🐍 Python-мост

ЛЮБОЙ Python-код прямо из Iron. 450 000+ библиотек PyPI.

💾 Базы данных

Встроенная поддержка SQLite. CREATE, INSERT, SELECT, DELETE.

🎲 Случайные числа

Функция rand для генерации случайных чисел.

Переменные

Объявление переменных в Iron — проще некуда. Используйте оператор %=%.

i %=% 1
name %=% Iron
pi %=% 3.14

При выводе переменной через put интерпретатор покажет её значение:

put i      // Выведет: 1
put name   // Выведет: Iron

Типы данных: Iron автоматически определяет тип. Можно явно указать int или str:

int age %=% 25
str city %=% Москва
💡 Лайфхак: Переменные можно использовать в любых командах — просто пишите их имя, и Iron сам подставит значение!

PUT — Вывод

Оператор put нужен для вывода на экран. Никаких скобок или кавычек!

put Hello World
put Привет, Iron!
put Сегодня 2025 год
💡 Совет: put автоматически выводит значения переменных. Просто напишите put name вместо print(name) в Python!

REQ / PREQ / REQS — HTTP запросы

REQ — GET запрос

// Простой GET запрос
req https://api.github.com

// GET запрос с переменной
url %=% https://api.local/users
req url

REQS — Массовые GET запросы (C++)

// Установите URL и запустите массовые запросы
https://api.local/test
reqs
// Интерпретатор спросит количество запросов

C++ отправит запросы со скоростью 500+ запросов/сек.

PREQ — POST запрос

// POST запрос с JSON
preq %=% {"name": "Iron", "version": "4.1"}

// POST запрос с переменной
data %=% {"user": "admin"}
preq %=% data

🆕 RAND — Случайные числа

Iron 4.1 поддерживает генерацию случайных чисел через команду rand:

// Случайное число от 1 до 10
rand 1 % 10

// Случайное число от 0 до 100
rand 0 % 100
🎮 Пример игры: Создайте генератор случайных чисел для угадайки или броска кубика!

MEMORY — Память

Оператор memory показывает все переменные и функции в памяти интерпретатора:

memory
// Выведет:
// ПЕРЕМЕННЫЕ: [{'key': 'i', 'value': 1}, ...]
// ФУНКЦИИ: [{'name': 'hello', 'value': 'put Привет'}]
🔧 Отладка: Используйте memory для проверки состояния переменных во время выполнения скрипта!

IF / ELSE — Условия

Iron поддерживает все основные операторы сравнения: =, >, <, >=, <=, !=.

Для тела условия используются префиксы: %i% (ИСТИНА) и %e% (ЛОЖЬ).

age %=% 18

if age > 16
%i% put Ты можешь получить паспорт!
%e% put Ты ещё слишком молод.
🎯 Сравнение с Python: 5 строк кода. В Iron — 3 строки!

%p% / %py%: / %py%/ — Python внутри Iron

%p% — одна Python-команда

// Вызов Python функции
%p% print('Hello from Python!')

// Создание GUI элемента
%p% btn = ctk.CTkButton(root, text="Кнопка")

%py%: / %py%/ — блок Python-кода

// Полноценная функция на Python внутри Iron
%py%:
def calculate():
    a = 10
    b = 20
    print(f"Сумма: {a + b}")
%py%/
🐍 Мощь Python: Через %p% и %py%: вы можете использовать ЛЮБУЮ библиотеку Python — NumPy, Pandas, TensorFlow, Requests, всё что угодно!

GUI — Графический интерфейс

Iron — один из немногих языков, где GUI можно создать за 3 строки!

Создание окна с текстом

init_root Мой проект
lbl Привет, мир!
close

Окно с кнопкой

init_root Кнопка
btn Нажми меня % %p%print("Привет!")
close

Калькулятор (полный пример)

init_root Калькулятор Iron

lbl Введите первое число:
%p% entry1 = ctk.CTkEntry(root)
%p% entry1.pack(pady=5)

lbl Введите второе число:
%p% entry2 = ctk.CTkEntry(root)
%p% entry2.pack(pady=5)

%py%:
def calculate():
    a = int(entry1.get())
    b = int(entry2.get())
    result_label.configure(text=f"Результат: {a + b}")
%py%/

%p% result_label = ctk.CTkLabel(root, text="")
%p% result_label.pack(pady=10)

btn Сложить % %p%calculate()
close
🎨 Совет: Всегда завершайте GUI-скрипт командой close!

🆕 Базы данных (SQLite)

Iron 4.1 имеет встроенную поддержку SQLite! Все команды интерактивны — интерпретатор подскажет, что вводить.

Создание таблицы

init_db
// Введите SQL: CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT, age INTEGER)

Добавление данных

insert
// Введите SQL: INSERT INTO users VALUES (1, 'Анна', 25)

Просмотр данных

select
// Введите SQL: SELECT * FROM users

Удаление данных

delete
// Введите SQL: DELETE FROM users WHERE id = 1
💾 Пример использования: Храните результаты тестов, пользователей, логи — всё в SQLite прямо из Iron!

🚀 Запуск из терминала (CLI)

Iron можно запускать из командной строки, передавая путь к файлу или папке со скриптами. После настройки алиаса iron вы можете запускать .iron файлы из любого места системы одной командой!

🐧 Linux (Debian / Ubuntu / Arch / IronEagleOS)

Способ 1: Алиас (рекомендуется)

# 1. Открой терминал и выполни:
echo 'alias iron="python3 ~/PycharmProjects/MainApp/main.py"' >> ~/.bashrc

# 2. Обнови конфигурацию:
source ~/.bashrc

# 3. Готово! Теперь Iron доступен отовсюду:
iron
iron scr/test.iron
iron scr/

Способ 2: Исполняемый файл (для продвинутых)

# 1. Создай файл:
sudo nano /usr/local/bin/iron

# 2. Вставь содержимое:
#!/bin/bash
python3 ~/PycharmProjects/MainApp/main.py "$@"

# 3. Сделай исполняемым:
sudo chmod +x /usr/local/bin/iron

# 4. Готово!
iron scr/test.iron

Способ 3: Добавление в PATH

# 1. Создай скрипт-обёртку:
nano ~/iron
# 2. Вставь содержимое:
#!/bin/bash
python3 ~/PycharmProjects/MainApp/main.py "$@"
# 3. Сделай исполняемым и добавь в PATH:
chmod +x ~/iron
echo 'export PATH="$HOME:$PATH"' >> ~/.bashrc
source ~/.bashrc

# 4. Готово!
iron scr/test.iron

🍎 macOS

Способ 1: Алиас (рекомендуется)

# 1. Открой терминал и выполни:
echo 'alias iron="python3 ~/PycharmProjects/MainApp/main.py"' >> ~/.zshrc

# 2. Обнови конфигурацию:
source ~/.zshrc

# 3. Готово!
iron
iron scr/test.iron
iron scr/

Способ 2: Добавление в PATH

# 1. Создай папку для скриптов (если ещё нет):
mkdir -p ~/bin

# 2. Создай скрипт-обёртку:
nano ~/bin/iron
# 3. Вставь содержимое:
#!/bin/bash
python3 ~/PycharmProjects/MainApp/main.py "$@"
# 4. Сделай исполняемым:
chmod +x ~/bin/iron

# 5. Добавь в PATH (добавь строку в ~/.zshrc):
echo 'export PATH="$HOME/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# 6. Готово!
iron scr/test.iron

🪟 Windows

Способ 1: Создание bat-файла

:: 1. Открой Блокнот и вставь:
@echo off
python3 C:\Users\ТвойПользователь\PycharmProjects\MainApp\main.py %*

:: 2. Сохрани как iron.bat в папке C:\Users\ТвойПользователь\

Способ 2: Добавление в PATH (Windows 10/11)

:: 1. Нажми Win + X → Система → Дополнительные параметры системы
:: 2. Переменные среды → Path → Изменить → Добавить
:: 3. Добавь путь: C:\Users\ТвойПользователь\
:: 4. Нажми ОК и перезапусти терминал

:: 5. Теперь можно запускать из любого места:
iron
iron scr\test.iron
iron scr\

Способ 3: PowerShell (альтернативный)

# 1. Открой PowerShell от администратора:
New-Item -Path $PROFILE -Force
notepad $PROFILE

# 2. Вставь строку:
function iron { python3 C:\Users\ТвойПользователь\PycharmProjects\MainApp\main.py $args }

# 3. Сохрани и перезапусти PowerShell
iron scr\test.iron

📂 FILE.R — Чтение файлов

Iron 4.2 позволяет читать содержимое любого файла прямо из скрипта одной командой. Это открывает возможности для парсинга логов, загрузки конфигураций или простого просмотра текста.

Синтаксис

file.r путь/к/файлу.txt

📋 Примеры использования

Чтение текстового файла

// Прочитать и вывести содержимое config.txt
file.r scr/config.txt

// Прочитать системный файл (Linux)
file.r /etc/hostname

Чтение другого .iron скрипта

// Загрузить и показать содержимое другого скрипта
file.r scr/old_version.iron

Комбинирование с другими командами

// Прочитать файл и сохранить результат в лог
@save
file.r scr/report.txt

// Прочитать все .iron файлы из папки через CLI
// Запустите в терминале:
// iron scr/

💡 Идеи для применения

📊 Парсинг логов

Читайте файлы логов вашего сервера прямо во время тестов API.

file.r /var/log/syslog

⚙️ Загрузка конфигураций

Храните URL и параметры в отдельных текстовых файлах.

file.r configs/api_url.txt

📝 Просмотр кода

Быстро просматривайте содержимое ваших старых скриптов.

file.r scr/backup.iron
💡 Совет: Команда file.r отлично сочетается с @save для создания отчётов. Просто прочитайте файл с данными и результат автоматически сохранится в scr/save.txt.
